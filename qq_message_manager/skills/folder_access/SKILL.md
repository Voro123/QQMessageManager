# 受控文件夹访问

这是一个能力 Skill。开启后，普通 AI 自动回复可以在最新实时入站消息涉及文件或项目操作时，根据关联名和描述判断最匹配的已授权文件夹，并调用本地受控文件工具；无法可靠判断时再向用户询问。

- 模型只能看到关联名、描述和读写权限，看不到真实绝对路径。
- 每个关联名直接代表其授权文件夹根目录。“项目主目录”“项目根目录”“授权目录”或“授权文件夹”均按该根目录处理；在根目录创建文件时直接使用文件名作为相对路径，不要再次索要真实路径或精确目录名。
- 代码库检索细节仅用于内部分析。最终回复默认问什么答什么，不主动展示文件路径、行号、工具过程、读取量、证据清单或源码；用户明确要求相关技术细节时再按需提供。
- 所有路径、可信 QQ、扩展名和读写权限都由本地权限层重新验证。
- 仅支持列目录、读取文本、搜索文本、文件信息、创建文件夹和写入文本。
- 不支持删除、移动、执行程序、运行代码、Shell、网络下载或访问授权目录之外的内容。
- 关闭本 Skill 后，不向模型提供授权信息，也不允许调用任何文件工具。
